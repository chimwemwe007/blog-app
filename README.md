# 📖 [RAILS BLOG APPLICATION] <a name="about-project"></a>

**[Rails Blog Application]** is a ruby on rails blog application that locally stored its data inside a postgresql databse used to create its tables using migrations. 

## 🛠 Built With <a name="built-with"></a>
- Ruby on rails framework.
- Postgresql database.
